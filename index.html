OVERVIEW

STUDIO IS A WEB-BASED MUSIC PRODUCTION ENVIRONMENT. ALL AUDIO IS PROCESSED LOCALLY IN YOUR BROWSER USING THE WEB AUDIO API. NO AUDIO DATA LEAVES YOUR MACHINE.

INSTRUMENTS

  • JUNO — POLYPHONIC SYNTHESIZER
  • SOUND BANK — FM SYNTHESIZER
  • SAMPLER — SAMPLER / BREAK SLICER
  • DRUMS — RHYTHM PROGRAMMER WITH MULTIPLE DRUM KITS

SAMPLER

16 VELOCITY-SENSITIVE PADS ACROSS 4 BANKS (A, B, C, D). LOAD ANY AUDIO FILE, AND THE SAMPLER AUTO-SLICES IT INTO PAD HITS USING TRANSIENT DETECTION.

  • BANKS C AND D AUTO-LOAD WITH RANDOM BREAKS FROM THE BUILT-IN LIBRARY ON STARTUP
  • FILE BROWSER FOR LOADING BREAKS FROM THE COLLECTION
  • ADJUSTABLE SENSITIVITY FOR SLICE DETECTION
  • RE-SLICE TO ADJUST SLICING IN REAL TIME
  • RECORD PAD SEQUENCES WITH QUANTIZE AND LOOP
  • EXPORT SEQUENCES AS JSON
  • DECK EXPORT SENDS LOOPS DIRECTLY TO SAMPLER PADS
  • PAD KEYS: 1234 (TOP ROW), QWER, ASDF, ZXCV (BOTTOM ROW)

PIANO ROLL

MIDI SEQUENCER FOR COMPOSING AND EDITING NOTE DATA ACROSS ALL INSTRUMENT CHANNELS.

  • CLICK TO PLACE NOTES, DRAG TO RESIZE
  • PER-CHANNEL NOTE EDITING WITH CHANNEL SELECTOR
  • VELOCITY LANE FOR PER-NOTE VELOCITY CONTROL
  • ADJUSTABLE BPM AND TIME SIGNATURE
  • SNAP TO GRID WITH MULTIPLE QUANTIZE VALUES
  • REAL-TIME RECORDING FROM QWERTY KEYBOARD OR MIDI CONTROLLER

MIXER

MIXING CONSOLE WITH PER-CHANNEL VOLUME FADERS, PAN, MUTE, SOLO, AND MASTER BUS CONTROLS. INCLUDES SEND EFFECTS (REVERB AND DELAY) WITH ADJUSTABLE LEVELS PER CHANNEL.

PEDALS

PER-CHANNEL PEDALBOARD WITH DRAG-AND-DROP EFFECTS CHAIN. AVAILABLE PEDALS:

  • OVERDRIVE, EQ, COMPRESSOR
  • CHORUS, PHASER, BITCRUSHER
  • VINYL (NOISE / WOW / FLUTTER / WEAR)
  • SP-303 LOFI, LOFI ECHO, LOFI REVERB
  • AW DRIVE

TAPE

TAPE MACHINE FOR RECORDING THE MASTER OUTPUT. ARM THE TRANSPORT, HIT RECORD, AND CAPTURE YOUR SESSION AS AN AUDIO FILE.

KEYBOARD SHORTCUTS

  • A-L QWERTY KEYS — PLAY NOTES ON THE FOCUSED INSTRUMENT (CHROMATIC MAPPING)
  • Z / X — OCTAVE DOWN / UP
  • SPACE — PLAY / STOP (PIANO ROLL AND SAMPLER)
  • R — START RECORDING (PIANO ROLL)
  • ESC — STOP / CLOSE MENUS
  • DELETE / BACKSPACE — DELETE SELECTED NOTES (PIANO ROLL)
  • CTRL+A — SELECT ALL NOTES (PIANO ROLL)
  • CTRL+D — DUPLICATE SELECTED NOTES (PIANO ROLL)

MIDI

CONNECT A MIDI CONTROLLER AND IT APPEARS IN THE TOP BAR. ROUTE EACH MIDI DEVICE TO ANY INSTRUMENT OR ALL INSTRUMENTS USING THE DEVICE DROPDOWN. THE KB SELECTOR ROUTES THE QWERTY KEYBOARD TO A SPECIFIC INSTRUMENT.

POPOUT

ANY INSTRUMENT OR UTILITY CAN BE POPPED OUT INTO ITS OWN WINDOW USING THE ARROW BUTTON IN THE TOP BAR. POPOUT WINDOWS STAY IN SYNC WITH THE MAIN SESSION.

SAVE / LOAD

SAVE SESSION EXPORTS A JSON FILE CONTAINING ALL INSTRUMENT STATES, PIANO ROLL DATA, DRUM PATTERNS, AND SAMPLER BANK CONFIGURATIONS. LOAD SESSION RESTORES A PREVIOUSLY SAVED FILE. ALL DATA IS STORED LOCALLY IN YOUR BROWSER.
